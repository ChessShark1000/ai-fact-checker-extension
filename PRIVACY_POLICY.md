# Privacy Policy for AI Fact Checker

**Last Updated:** December 3, 2025

## Overview
AI Fact Checker is a Chrome extension that provides instant fact-checking for text selected on web pages.

## Data Collection
- **What we collect:** Text that you manually highlight and select for fact-checking
- **How we use it:** Selected text is sent to our backend server (ai-factchecker.com) and then to OpenAI's API for AI-powered analysis
- **What we don't collect:** We do not collect, store, or track browsing history, personal information, or any data beyond the text you explicitly select for fact-checking

## Data Storage
- Selected text is processed in real-time and **not stored permanently**
- Chat conversations exist only during your active session and are cleared when you close the detailed view
- No user accounts, profiles, or persistent data storage

## Third-Party Services
- We use OpenAI's API to perform fact-checking analysis
- OpenAI's privacy policy: https://openai.com/policies/privacy-policy
- All data transmission uses HTTPS encryption

## Data Sharing
- We do not sell, rent, or share your data with third parties
- Selected text is only shared with OpenAI for the purpose of fact-checking analysis

## Your Rights
- You control what text is sent for analysis by choosing what to highlight
- You can uninstall the extension at any time to stop all data processing

## Changes to This Policy
We may update this privacy policy from time to time. Changes will be posted on this page.
